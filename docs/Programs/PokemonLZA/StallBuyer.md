# Stall Buyer

## Program Description

Purchase berries or mints from stalls.


### Setup of Settings

**Game Settings:**

1. Text Speed: Fast

## Instructions

**Switch Settings:**

1. Screen size: Must be 100% within the Switch settings
2. [Switch 2: All HDR options must be disabled.](../NintendoSwitch/Switch2Notes.md#switch-2-hdr-may-be-problematic)

**Program Settings:**

1. Video Resolution: 1080p or higher

**Game Settings:**

1. Text Speed: Fast


### Instructions

1. Go to the stall that you wish to purchase something from
<img src="images/StallBuyer-Location-1.png">
<img src="images/StallBuyer-Location-2.png">
2. Start the program in-game. The program will stop after purchasing the assigned amount of items. The program fails safely if the player is out of money, but try to ensure the player has enough money at the start of the program.

## Options

### Number to Purchase:

The number of items you want to purchase.

### Go Home when Done:

Go to the Switch Home to idle when finished.
