# Berry Buyer

## Program Description

Purchase EV reducing berries from stalls.

## Settings

**Game Settings:**

1. Text Speed: Fast

## Instructions

1. Go to the stall that sells all the EV reducing berries
<img src="images/BerryBuyer-Location-1.png">
<img src="images/BerryBuyer-Location-2.png">
2. Start the program in-game. The program will stop after purchasing the assigned amount of berries. The program fails safely if the player is out of money, but try to ensure the player has enough money at the start of the program.

## Options

### Berry to Purchase:

The EV reducing berry you want to purchase, each berry reduces 10 EVs in a respective stat:
- Pomeg (HP)
- Kelpsy (Attack)
- Qualot (Defence)
- Hondew (Special Attack)
- Grepa (Special Defence)
- Tamato (Speed)

### Number to Purchase:

The number of berries you want to purchase.

### Go Home when Done:

Go to the Switch Home to idle when finished.
